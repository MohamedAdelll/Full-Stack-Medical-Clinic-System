## Getting Started

Database running on port 5432 and server running on port 5000

### Prerequisites

You should have [mySQL](https://dev.mysql.com/downloads/mysql/) installed and [Python](https://www.python.org/downloads/) installed on your comuter

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
2. Run DB.py only once

3. Run main.py
